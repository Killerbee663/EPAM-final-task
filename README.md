SauceDemo Automation Task (WDIO)

This repository contains a test automation suite for the [SauceDemo](https://www.saucedemo.com/) website. It covers basic E2E checkout flows (UC-1) and data-driven login validation (UC-2) using **WebdriverIO** and **Mocha**.

## Tech Stack
Framework: WebdriverIO (v9)
Test Runner: Mocha
Pattern: Page Object Model (POM)
Reporting: Mochawesome
Coverage: c8
Linting: ESLint (Flat Config)

Getting Started

1. Prerequisites
Ensure you have **Node.js** installed (v18 or higher recommended). 

2. Installation
Clone the project and install the dependencies:
```bash
npm install
npm test
npm run lint
